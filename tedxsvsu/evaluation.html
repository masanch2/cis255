Presentation Very informative. This page contains a lot of information right off the bat and each page is organized into readable snippets. Nothing terribly eye-catching on the main page, but looking further one can easily determine the nature of the event. Has a 'modern' feel. There is a more graphical focus on most pages, however all three sub-pages provide relevant detail. Loading each page transitions nicely. The only distraction I found was that it had a slightly generic feel, opposed to maintaining an 'SVSU' theme. I aimed for a clean layout with a short summary of each page available on a single page. I don't get the feel my theme (as-is) really portrays the event type, but I did try to include SVSU in the design along with original photos.
Accessibility The main page content answers initial questions, and the menu provides total site navigation for the curious visitor. Provides a concise menu and descriptions of the nature of the event, speakers and how to participate. Once again, my goal here was to present each of the three pages in summary below an eye-catching visual that would represent SVSU.
